Encapsulation

Encapsulation: The grouping of data and methods that operate on this data into a single unit

Purposes:

  • Allows for common and shared functionalities for different groups of data
  • Protects data from being modified by external users
  • Encapsulation is usually done within a Class
  • The actual data structures used and method implementation are hidden from the user
  • Different levels of encapsulation
