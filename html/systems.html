Real Time Operating Systems (RTOS)

Real Time Operating System (RTOS): operating system used for applications requiring real-time data-processing

General purpose OS vs RTOS: THE SCHEDULER

  • RTOS depends on requests being responded to in a set amount of time
  • The scheduler's behaviour must be known ahead of time
  • This can be done by setting priorities to the executing threads

What is an Embedded System

Embedded System: A mini computer!

  • Think of it as a mini computer composed of hardware and software components
  • Acts as a smaller controller to a larger system

Microcontrollers

Microcontroller: Smaller unit within an embedded system

  • Integrated circuit containing its own processor and memory
